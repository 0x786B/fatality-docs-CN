## vec2

此类型是渲染系统中使用的 2D 向量。

## __call

[![构造函数][这是此类型的构造函数定义。]rw]

创建一个新的 2D 向量实例。

**参数**

*1. 默认向量 (0, 0)*

无。

*2. 单个值*

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `value` | `float` | X 和 Y 坐标。 |

*3. XY 值*

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `x` | `float` | X 坐标。 |
| `y` | `float` | Y 坐标。 |

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `vec2` | 新向量。 |

**示例**

```lua
local vec = draw.vec2(5, 10);
```
