## game_event_t

描述游戏事件。

## get_name

[![方法][这是一个方法，必须使用冒号 (:) 调用。]rw]

返回事件名称。

**参数**

无

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `string` | 事件名称。 |

**示例**

```lua
if event:get_name() == 'player_hurt' then
    -- ...
end
```

## get_bool

[![方法][这是一个方法，必须使用冒号 (:) 调用。]rw]

**参数**

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `key` | `string` | 条目键。 |

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `bool` | 条目值。如果未找到该键，则返回 `false`。 |

**示例**

```lua

```
