## vector

这是一个通用的3D向量类型(x, y, z)。

> 此类型支持运算符操作,如 `+`, `-`, `/`, `*` 和 `==`。

## x

[![字段][这是一个普通字段,必须使用点(.)访问]rw]

类型: `float`

X坐标。

## y

[![字段][这是一个普通字段,必须使用点(.)访问]rw]

类型: `float`

Y坐标。

## z

[![字段][这是一个普通字段,必须使用点(.)访问]rw]

类型: `float`

Z坐标。

## __call

[![构造函数][这是此类型的构造函数定义]rw]

构造一个向量。

**参数**

*1. 默认向量 (0, 0, 0)。*

无。

*2. 单个值。*

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `value` | `float` | X和Y坐标。 |

*3. XYZ值。*

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `x` | `float` | X坐标。 |
| `y` | `float` | Y坐标。 |
| `z` | `float` | Z坐标。 |

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `vector` | 向量。 |

**示例**

```lua
local vec = vector(5, 25, 12);
```

## is_zero

[![方法][这是一个方法,必须使用冒号(:)调用]rw]

如果该向量在容差范围内则返回`true`。

**参数**

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `tolerance` | `float` | 最大允许容差。默认为`0.01`。 |

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `bool` | 如果值在`-tolerance`和`tolerance`之间则返回`true`。 |

**示例**

```lua
local is_zero = vec:is_zero(); -- 使用默认容差
local is_zero_custom = vec:is_zero(0.1); -- 自定义容差
```
