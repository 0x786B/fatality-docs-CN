## observer_mode_t

此枚举表示游戏中可用的观察者模式。

## none

[![Field][此字段是一个普通字段，必须使用点(.)来访问]rw]

表示无观察者模式（不在观察任何人）。

## fixed

[![Field][此字段是一个普通字段，必须使用点(.)来访问]rw]

表示固定摄像机位置。

## in_eye

[![Field][此字段是一个普通字段，必须使用点(.)来访问]rw]

表示从被观察玩家的视角观察。

## chase

[![Field][此字段是一个普通字段，必须使用点(.)来访问]rw]

表示跟随被观察玩家的追踪摄像机。

## roaming

[![Field][此字段是一个普通字段，必须使用点(.)来访问]rw]

表示自由漫游摄像机。

## directed

[![Field][此字段是一个普通字段，必须使用点(.)来访问]rw]

表示自动导向的摄像机，通常由游戏或服务器控制。
