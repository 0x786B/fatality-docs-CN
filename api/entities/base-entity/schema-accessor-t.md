## schema_accessor_t

此类型表示一个特殊的结构，用于引用实体对象中的某个字段。

> 你可以使用 `type()` 函数检查返回的类型。

> 切勿在事件作用域之外的任何地方存储此类型的实例，因为实体可能会被移除。

## get

[![方法][此字段是一个方法，必须使用冒号(:)调用。]rw]

返回值。

**参数**

无。

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `<type>` | 值。 |

**示例**

```lua
local health = player.m_iHealth:get();
```

## set

[![方法][此字段是一个方法，必须使用冒号(:)调用。]rw]

设置值。

**参数**

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `value` | `<type>` | 值。 |

**返回值**

无。

**示例**

```lua
player.m_iHealth:set(50); -- 实际上不会对生命值产生任何影响
```
